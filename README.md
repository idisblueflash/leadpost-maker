# leadpost-maker

把你写好的文章（比如 newsletter），自动改写成适合发到 Facebook 的内容。

## 这是做什么的

写文章和发社群是两件事。一篇适合放在 newsletter 里的长文，原样搬到 Facebook 上往往没人看完——开头不够抓人，节奏太慢，结尾也没有引导。每发一篇都手动重写一遍，太累。

这个项目把「改写」这件事变成一键操作。你放一篇文章进去，它帮你产出两种东西：

- **引流帖**：一段精简、好读、能把人勾回你 newsletter 的 Facebook 文案，还附三个标题让你挑。
- **轮播图**：一组 9 张的方形图片（Facebook 多图帖），把文章重点做成可以一张张滑的图。

## 怎么用

1. 把文章原文放进 `input/` 文件夹（一个 `.md` 文件）。

2. 在 Claude Code 里说一句话就行：

   - 想要引流帖：**「把这篇转成 Facebook 帖」**（或 `/leadpost`）
   - 想要轮播图：**「把这篇做成轮播图」**（或 `/carousel`）

   你也可以直接说「转最新那篇」，不用指文件名。

3. 做轮播图时，它会先给你一张九宫格预览，让你挑喜欢哪几张、按什么顺序，再正式出图——整个过程你说了算。

4. 结果都放在 `output/` 里，按文章名分好文件夹：
   - `output/<文章名>/post.md` —— 引流帖
   - `output/<文章名>/carousel/` —— 轮播图

5. 不满意随时说，比如「换个标题」「再短一点」「这张图重做」，它会改。

## 想调整改写风格

帖子的写法、标题类型、图片的视觉风格这些规则，都写在 `.claude/skills/` 文件夹里，想改可以直接改那里的文件。日常使用不需要碰。
