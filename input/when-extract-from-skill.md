# 什么时候该把计算从 Skill 里拆出来

你以为 AI 在算数，其实它在玩成语接龙. 

就拿我设计的「贝叶斯 Skill」来说，让它预测明天会下雨么。每次给它一点新内容：闻到了泥土味，看到了石头出汗、墙壁返潮，家养的金鱼把头浮出了水面，看到了蚂蚁正搬家。它每次都会预估一个概率，然后依照贝叶斯算法给出后面的概率。

我刚开始的时候偷懒，让 AI 自己计算概率。它算出来分数看着像那么回事，我也没深究。直到有一天我给 Skill 加上了测试，才发现这个问题。

有篇论文提到，让 LLM 计算贝叶斯，它只是在「猜」数字。正确率只有 20%，哪怕是有「思维链」的推导模式，正确率也只有 65% 。可是换成脚本计算的正确率可以高达 97%。

把这种 LLM 不擅长的计算拆成脚本才是正路。
